## 시작 가이드

### Installation

```
$ git clone https://github.com/MyHomeYogiyo/MyHomeYogiyo.git
$ cd MyHomeYogiyo
```

### Backend

```
$ cd BE/MyHomeYogiyo
$ ./mvnw spring-boot:run
```

### Frontend

```
$ cd FE
$ npm install 
$ npm run dev
```
