# 🏡 내집은 요기요 (MyHomeYogiyo)

SSAFY 10기 1학기 관통프로젝트

부동산 정보 제공 웹 서비스 "내 집은 요기요"

개발기간 : 23.11.16 ~ 23.11.24

---

## 프로젝트 소개

사용자에게 특정 지역의 부동산 매매정보를 제공함으로써 더 나은 의사결정을 도와주는 웹 서비스

### ⚡ 지역별 부동산 매매정보 검색 기능

    - 매물 좋아요

    - 부동산 매매정보 연도별 조회 기능

    - 연도별 거래량 분석 그래프 조회 기능

### ⚡ 게시판 기능

    - 공지사항 CRUD
